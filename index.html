Miniprovistor

Deploy a Digital Ocean virtual private server and install OpenBazaar hassle free!
Manage your OpenBazaar node easily once it's provisioned.

Start Here

To get started, all you need to do is join Digital Ocean and create an API token (super easy to do). The GIF below should help you find that. Currently, provisioning an OpenBazaar node is a 2-step process, but in future all you'll need to do is add the API token and press a button.



Security: the miniprovistor server never sees your Digital Ocean API access token (check the source if you don't believe me). However, any node that you create using miniprovistor will be temporarily root-accessible to the server. After installation, root access is automatically removed.
